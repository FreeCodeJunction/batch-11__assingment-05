Question-1: What are the different ways to select an element in the dom?

There are many different ways to select an element in the dom. To select a single element we can use the "getElementById()" and "querySelector()" method DOM provides. The "getElementById()" method select an element with their id's and the "querySelector()" method selects an element by using a css selector. To select multiple elements in the dom, there are getElementsByClassName() , getElementsByTagName(), querySelectorAll() method .

Question-2: What is the difference between innerHTML, innerText, and textContent ?

"innerHTML": It returns or sets the html content of an element including tags , spaces , attributes. If we sets an element's content with innerHTML , the content will act or behave like html.
"innerText": It returns or sets the text content of an element. It doesn't show the spaces , tags , anything other that the visible text content.
"textContent": It returns or sets the text content of an element . If the elements contains multiple spaces , it also shows them.

Question-3: What is event delegation in the DOM?

When we have a situation where we need to attach event listeners to multiple child elements who are descendent of the same parent , instead of attaching event listeners to every single one of them we add an event listener to the parent element and based on event bubbling , we determine which child triggered the event.

Question-4: What is event bubbling in the DOM?

Event bubbling in javascript is when an event listener is triggered , how it propagates through its ancestor. When an event is triggered by an element, the event is bubbled up from the targeted element to its parent and ancestor elements until it reaches the root or the html element.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create element with the document.createElement() method. Then to add the element into the dom, we can use the append or the appendChild method. Also there are so many methods to add html content or element in the dom like the innerHTML, insertAdjacentElement , insertAdjacentHTML etc. And to remove an element from the dom, the remove() , removeChild() method is used. We can also use the innerHTML method to empty en html element.
